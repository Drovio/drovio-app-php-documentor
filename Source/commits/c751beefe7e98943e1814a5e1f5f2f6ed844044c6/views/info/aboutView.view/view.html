PHP Documentor using Redback Documentation Engine

This application helps generate documentation for php classes.

Follow the directions to create your documentation:

  1. 1. Write/Paste your code in the left panel.
  2. 2. Hit the documentation button on the right bottom corner to show the documentation form.
  3. 3. Fill all the information you need about the given class code.
  4. 4. Press the play button on the top left corner to get your documented code.

The script prints on the right panel a full-commented code of the class given and the documentation xml model as the engine parses it.

At this moment the application parses only classes (there must be a class visible) and functions must have the access modified (public/protected/private).
